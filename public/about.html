About VideoStream

A self-hosted video streaming platform built as a hands-on DevOps learning project. Every component — Node.js, Nginx, Docker, and eventually Kubernetes — is deployed and configured manually to build real production skills.

Tech Stack

🟢
Node.js + Express
Backend server. HTTP range streaming, session auth, file management.
⚡
Nginx
Reverse proxy. Rate limiting, caching, security headers, location routing.
🐳
Docker
Containerized deployment. Coming in Phase 4.
☸️
Kubernetes
Orchestration, autoscaling, load balancing. Coming in Phase 6.
🔧
dnsmasq
Local DNS. Resolves videostream.local across the home network.
🖥
Proxmox VE
Homelab hypervisor. Ubuntu 22.04 VMs for staging and testing.

Build Phases

Project Roadmap
1 Bare Metal Node.js Express server, video streaming, range requests Done

3 Nginx Deep Dive Rate limiting, caching, security headers, error pages Current
4 Dockerize Dockerfile, docker-compose, container networking Next
5 Nginx + Docker Nginx proxying to containers, volume mounts Planned
6 Kubernetes Deployments, services, ingress, autoscaling Planned
